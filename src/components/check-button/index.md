# CheckButton 组件
> 勾选组件。

- 用于项目勾选按钮场景

## 属性

| 属性            | 说明         | 类型          | 默认值      |
| -------------- | ----------- | ----------- | ----------- |
| className      | css类名      | `string`    | -  |
| checked        | 是否选中      | `boolean`   | -   |
| showIcon       | 是否显示图标（不显示图标时，选中是圆圈样式；显示图标时，选中是圆圈加打勾样式）   | `boolean`   | `true`     |
